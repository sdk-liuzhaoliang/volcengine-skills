# skills/

存放所有 skill，一个 skill 一个子目录，目录名以 `volcengine-` 开头。

```
skills/
└── volcengine-<xxx>/
    ├── SKILL.md          # 必填
    ├── references/       # 可选：长文档 / API 参考
    ├── scripts/          # 可选：可执行脚本
    └── assets/           # 可选：模板 / 资源
```

新增 skill 前请先读 [`../docs/contributing-guide.md`](../docs/contributing-guide.md)。
